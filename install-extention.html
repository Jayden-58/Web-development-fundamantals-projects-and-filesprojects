how to install and use VS Code's Live Server Extension

install | use | video

Installation Instructions

  1. In visual studio code, click the extentions icon in the left column tool bar
  2. In the search bar that appears, type live server.
  3. Select the Live server extention, it should be the top choice
  4. In the page for Live server that appears, click the green install button.

Usage instructions:

  • Visual studio code must be open to a folder to use live server
  • In vs code's explored, right click (windows) or control+click (macOS) on an HTML file. From the content menu that appears, select Open with live server.
